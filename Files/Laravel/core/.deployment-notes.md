# Notas de Implantação - Mobilidade Urbana

## Configuração Local

### Pré-requisitos
- PHP 8.2+
- MySQL/MariaDB
- Composer
- Node.js (opcional, para assets)

### Passos de Instalação Local
1. **Banco de dados criado**: `mobilidade_urbana`
2. **Dependências instaladas**: `composer install` (concluído)
3. **Arquivo .env configurado** com:
   - APP_NAME="Mobilidade Urbana"
   - APP_ENV=local
   - APP_DEBUG=true
   - APP_LOCALE=pt_BR (Português do Brasil)
   - APP_FALLBACK_LOCALE=pt_BR
   - DB_DATABASE=mobilidade_urbana
   - DB_USERNAME=root
   - DB_PASSWORD= (vazio para XAMPP)

### Executar Aplicação Localmente
```bash
cd c:\Projetos\MobilidadeUrbana\Files\Laravel\core
php artisan serve
```

Acesse: http://localhost:8000

### Credenciais de Admin
- Email: admin@site.com
- Senha: (verificar no banco ou resetar)

## Deploy no Render

### Preparação
1. **Criar conta no Render**: https://render.com
2. **Criar banco de dados MySQL** no Render
3. **Criar Web Service** apontando para o repositório Git

### Arquivos de Configuração Criados
- `render.yaml` - Configuração do Render (blueprint)
- `Procfile` - Comando de inicialização
- `.deployment-notes.md` - Este arquivo

### Variáveis de Ambiente no Render
Configure no painel do Render:
```
APP_NAME=Mobilidade Urbana
APP_ENV=production
APP_DEBUG=false
APP_KEY=(gerar com: php artisan key:generate --show)
APP_URL=https://seu-app.onrender.com
APP_TIMEZONE=America/Sao_Paulo
APP_LOCALE=pt_BR
APP_FALLBACK_LOCALE=pt_BR
APP_FAKER_LOCALE=pt_BR

DB_CONNECTION=mysql
DB_HOST=(fornecido pelo Render MySQL)
DB_PORT=3306
DB_DATABASE=mobilidade_urbana
DB_USERNAME=(fornecido pelo Render)
DB_PASSWORD=(fornecido pelo Render)
```

### Build Command no Render
```bash
composer install --no-dev --optimize-autoloader --ignore-platform-req=ext-gmp
php artisan config:cache
php artisan route:cache
php artisan view:cache
```

### Start Command no Render
```bash
php artisan serve --host=0.0.0.0 --port=$PORT
```

### Após Deploy
1. **Importar banco de dados**: Execute o SQL do arquivo `install/database.sql` no banco MySQL do Render
2. **Configurar domínio**: Configurar APP_URL com o domínio real
3. **Testar aplicação**: Verificar login admin e funcionalidades

### Observações
- O projeto usa MariaDB 10.4.32 localmente
- Idioma configurado: **Português do Brasil (pt_BR)** - definido como padrão
- Idioma secundário: Português (pt), Inglês (en), Espanhol (es), Francês (fr), etc.
- Extensão GMP não está disponível localmente (ignorada com flag)
- Timezone: America/Sao_Paulo

### Estrutura do Projeto
- **Laravel**: Backend API e admin
- **Flutter**: Apps mobile (Driver e Rider)
