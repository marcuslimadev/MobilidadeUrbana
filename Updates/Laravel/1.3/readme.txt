HOW TO UPDATE :

1. Upload the file named "Files/Update.zip" to the root directory of your server.
2. Extract its contents directly into the root directory.
3. Import the "Files/update.sql" file into your database.

Ensure that all steps are completed accurately to apply the update successfully.

Note:
Before proceeding, please ensure that you have a full backup of your files and database.

After completing the update, kindly refresh your general settings by navigating to the General Settings section in the admin panel.

This version includes an update to the driver vehicle verification process. As a result, all previously unverified/verified drivers will be automatically affected by this change.
Please make sure to re-verify your drivers from the admin panel to ensure proper functionality.
